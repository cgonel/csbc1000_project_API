## Setup Guide

Install the dependencies

```
composer install
```

Migrate the migrations 

```
php artisan migrate
```

Start the server

```
php artisan serve
```

Note: if you want to populate the database with dummy data.

run `php artisan db:seed`

For updating consposer if required

run 'composer update'
